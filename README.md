# Fusion-360 Door hinge
My second project in the world of 3D consists of a door hinge that got broken in one of the furniture I had in my house. Since this hinge was really old and I was not able to find it in any tool shop I decided to make my own. It took me several tries to get the right mesurments with the Vernier caliper. Although I was not able to get the best and most fitting object designed I still 3D printed it and applied on the door. Still it was great to learn and grasp the concepts like revolve, sketch, cut, join and align.
